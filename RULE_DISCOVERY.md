## 已确认发现

### 1. 不能用单纯面积公式解释全部报价

- `5*10 覆亮膜 + 彩色印刷+烫金/银`：`326 / 345 / 579`
- `6*10 覆亮膜 + 彩色印刷+烫金/银`：`413 / 436 / 588`
- 两者面积接近，但价格跳变明显。
- 结论：不能只按面积做连续插值。

### 2. 不能用“统一 5 取整”解释全部报价

- `5*5 / 5*6 / 5*7 / 5*9 覆哑膜 + 彩色印刷+烫金/银` 同价：`341 / 355 / 620`
- 但 `6*6`：`356 / 374 / 651`
- `6*7`：`365 / 385 / 667`
- `6*8`：`374 / 396 / 685`
- `6*9`：`433 / 457 / 695`
- 结论：不是简单的“宽高都按 5 取整”。

### 3. 某些尺寸段会完全同价，说明存在拼版档位

- `3*5`：`41 / 46 / 64`
- `4*5`：`41 / 46 / 64`
- `5*5`：`69 / 81 / 110`
- `7*5`：`69 / 81 / 110`
- `10*5`：`102 / 121 / 173`
- 结论：价格更像按排版/拼版档位跳变，而不是按尺寸平滑变化。

### 4. 不同工艺大概率走不同规则

- `覆亮膜 + 彩色印刷+烫金/银` 与 `覆哑膜 + 彩色印刷+烫金/银` 的跳变边界不一致。
- `覆哑膜 + 纯烫金` 也表现出独立档位。
- 结论：不能假设所有工艺共用同一套尺寸公式。

### 5. `覆哑膜 + 纯烫金` 至少存在“同档同价”现象

- `3*7 覆哑膜 + 纯烫金`：`255 / 268 / 458`
- `5*5 覆哑膜 + 纯烫金`：`255 / 268 / 458`
- 结论：这两个尺寸在该工艺下落在同一报价档。

### 6. `模切 - 覆哑膜 - 彩色印刷+烫金/银` 新补样本继续支持当前档位表

- 新补基线：

  - `5*6`：`296 / 308 / 539`
  - `5*7`：`296 / 308 / 539`
  - `6*7`：`317 / 334 / 580`
  - `6*8`：`325 / 344 / 595`
  - `6*10`：`430 / 453 / 610`
- 结论：
  - `5*5 ~ 5*7` 同档同价再次被确认。
  - `6*8` 已被基线补齐，不再只是从相邻规则推断。
  - 在这组工艺上，`模切` 当前继续可按与 `裁切` 同价处理。

### 7. `覆哑膜 + 彩色印刷+烫金/银` 已确认存在更小尺寸低档

  - `3*3`、`3*5`、`4*4`、`4*5` 仍属于未补样本，暂不应硬并档。

## 当前最合理判断

- 这套报价底层更像“拼版档位规则”而不是“面积连续公式”。
- 现阶段基于完整基线样本，已经可以先按“工艺分组 + 尺寸档位 + 数量档价格”收口实现。
- `裁切` 与 `模切` 当前按同价处理。
- `纯烫金` 与 `烫镭射银` 当前按同价处理。
