  - 覆哑膜 + 彩色印刷+烫金/银
  - 覆哑膜 + 纯烫金

## 已确认发现

### 1. 不能用单纯面积公式解释全部报价

- `5*10 覆亮膜 + 彩色印刷+烫金/银`：`326 / 345 / 579`
- `6*10 覆亮膜 + 彩色印刷+烫金/银`：`413 / 436 / 588`
- 两者面积接近，但价格跳变明显。
- 结论：不能只按面积做连续插值。

### 2. 不能用“统一 5 取整”解释全部报价

- `5*5 / 5*6 / 5*7 / 5*9 覆哑膜 + 彩色印刷+烫金/银` 同价：`341 / 355 / 620`
- 但 `6*6`：`356 / 374 / 651`
- `6*7`：`365 / 385 / 667`
- `6*8`：`374 / 396 / 685`
- `6*9`：`433 / 457 / 695`
- 结论：不是简单的“宽高都按 5 取整”。

### 3. 某些尺寸段会完全同价，说明存在拼版档位

- `3*5`：`41 / 46 / 64`
- `4*5`：`41 / 46 / 64`
- `5*5`：`69 / 81 / 110`
- `7*5`：`69 / 81 / 110`
- `10*5`：`102 / 121 / 173`
- 结论：价格更像按排版/拼版档位跳变，而不是按尺寸平滑变化。

### 4. 不同工艺大概率走不同规则

- `覆亮膜 + 彩色印刷+烫金/银` 与 `覆哑膜 + 彩色印刷+烫金/银` 的跳变边界不一致。
- `覆哑膜 + 纯烫金` 也表现出独立档位。
- 结论：不能假设所有工艺共用同一套尺寸公式。

### 5. `覆哑膜 + 纯烫金` 至少存在“同档同价”现象

- `3*7 覆哑膜 + 纯烫金`：`255 / 268 / 458`
- `5*5 覆哑膜 + 纯烫金`：`255 / 268 / 458`
- 结论：这两个尺寸在该工艺下落在同一报价档。

## 当前最合理判断

- 这套报价底层更像“拼版公式”而不是“面积公式”。
- 更可能的计算结构：

```text
总价 = 固定开机/制版费
    + 印刷费(由拼版数决定)
    + 覆膜费(由拼版数决定)
    + 烫金费(由拼版数 + 烫金档决定)
```

- 其中关键中间量更可能是：

```text
可排个数 = floor(版宽 / (成品宽 + 出血/刀缝))
        * floor(版高 / (成品高 + 出血/刀缝))

所需版张数 = ceil(数量 / 可排个数)
```

- 再由版张数去推导不同数量档的价格。

## 当前实现上的警告

- 现阶段代码中的规则仍然只是“阶段性拟合”。
- 这些拟合可以覆盖部分已知样本，但还没有证明等价于真实业务规则。
- 后续不要再默认：
  - 只按面积算
  - 只按 5 取整
  - 所有工艺共用同一套公式
